# 如何針對 Azure 計費和訂用帳戶問題建立支援票證

本文提供建立 Azure 支援票證所需的資訊。

>[AZURE.NOTE]僅 Azure 帳戶管理員有權存取帳戶中心。

1. 登入 [Azure 入口網站](https://portal.azure.com/)。選取 [說明 + 支援] > [新增支援要求]。

	![helpandsupportbutton](./media/billing-how-to-create-billing-support-ticket/helpandsupport.png)
2. 在 [基本] 刀鋒視窗中，選擇問題類型、訂用帳戶和支援方案。
3. 在 [問題] 刀鋒視窗中提供下列資訊：

 * 嚴重性。選擇代表要求嚴重性的選項。如需嚴重性類型的詳細資訊，請參閱 [Microsoft Azure Incident Severity Table (Microsoft Azure 事件嚴重性資料表)](http://support.microsoft.com/gp/AzureSevDetails)。
 * 問題類型。選擇問題類型。
 * 詳細資料。為您的要求提供事件標題，然後在文字方塊中提供必要資訊，並附上任何有關要求的其他資訊。
 * 時間範圍 (選用)。可能的話，請提供問題最新發生的日期、時間和時區資訊。
 * 使用「檔案上傳」工具，將相關文件附加至您的要求。
4. 在 [連絡資訊] 刀鋒視窗中，確認您的連絡資訊和慣用的連絡方法，然後按一下 [建立]。

Azure 支援代表將會根據合約條款來與您聯絡。

您也可以從 Azure 支援中心網站或 Azure 帳戶中心提交支援要求︰

 * 若要從 [Azure 支援中心網站](https://azure.microsoft.com/support/options/)提交支援票證，請按一下 [取得支援]。
 * 如需從 [Azure 帳戶中心](https://account.windowsazure.com/Subscriptions)提交支援票證，請選取訂用帳戶和 [連絡 Microsoft 支援服務]。
