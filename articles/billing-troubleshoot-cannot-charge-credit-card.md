# 我收到一封電子郵件通知我服務可能會中斷

如果我們因為某些原因無法處理您的付款，則您可能會收到一封電子郵件，並含有類似如下的訊息：

**我們已無法針對您的訂用帳戶向您的信用卡收取費用。若要避免服務發生中斷，請更新您的付款資訊。**

如果您的帳戶已過期，也可能會發生此情況。如果是因為帳戶過期，請參閱本文的指示，使用信用卡或發票付款：[為什麼會收到 Azure 訂用帳戶有逾期未付帳款的通知](billing-azure-subscription-past-due-balance.md)。

若本文中未提及您的 Azure 問題，請造訪 [MSDN 及 Stack Overflow](https://azure.microsoft.com/support/forums/) 上的 Azure 論壇。您可以在這些論壇上，或前往 Twitter 的 @AzureSupport 上張貼您的問題。此外，您也可以選取 [Azure 支援](https://azure.microsoft.com/support/options/)網站上的 [取得支援]，來提出 Azure 支援要求。

如果您需要建立支援票證的協助，請參閱[如何針對 Azure 計費和訂用帳戶問題建立支援票證](billing-how-to-create-billing-support-ticket.md)。
