此設定可讓您指定要用於此虛擬網路之名稱解析的 DNS 服務。它不會建立 DNS 伺服器。

1. 在您虛擬網路的 [設定] 頁面上，瀏覽到 [DNS 伺服器] 並按一下以開啟 DNS 伺服器刀鋒視窗。
2. 在 [DNS 伺服器] 頁面中的 [DNS 伺服器] 之下，選取 [自訂 DNS]。
3. 在 [主要 DNS 伺服器] 欄位中，輸入要用於名稱解析之 DNS 伺服器的 IP 位址。
4. 按一下刀鋒視窗頂端的 [儲存] 來儲存您的組態。
